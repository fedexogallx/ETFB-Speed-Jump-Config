# Speed & Jump GUI

A clean and customizable Speed & Jump interface for Roblox.

### Features

* Change **CurrentSpeed** and **JumpUpgrade** using an easy-to-use interface.
* Values are applied only when the **Apply** button is pressed.
* **Reset** button restores the default values.
* Live synchronization with Player Attributes.
* Smooth draggable window.
* Minimize and close buttons.
* Semi-transparent modern design.
* Animated rainbow border.
* Customizable keybind to show or hide the GUI.
* Click the keybind button, then press any keyboard key to assign a new hotkey.
* The hotkey is ignored while typing in text boxes to prevent accidental toggling.

Designed for projects that use Player Attributes instead of directly modifying the Humanoid, making it easy to integrate into custom movement systems.
